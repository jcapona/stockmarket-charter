# stockmarket-charter

Displays a graph with the recent trend lines for stocks selected by the users.

Uses Node.js + MongoDB

## Test it

Check it out live @ [stockmarket-charter.herokuapp.com](http://stockmarket-charter.herokuapp.com/)
